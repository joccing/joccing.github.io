Class: ctTable

ctTable(row, col, rowlast, collast, sheetname, UseCache) → {Object}

new ctTable(row, col, rowlast, collast, sheetname, UseCache) → {Object}

Create an abstraction of any range of a Google spreadsheet so that read and writes to it are much faster when the cache is turned on. All methods in ctTable are wrapped and prefixed with 'ct'. We call this a CachedTable.
Parameters:
Name Type Description
row integer Starting row number of the section
col integer Starting col number of the section
rowlast integer Ending row number of the section
collast integer Ending column number of the section
sheetname string Name of the worksheet containing this section
UseCache boolean Set to true if you want to use the cache (recommended)
