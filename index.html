Gaslight

Gaslight is a Google Appscript library that provides useful classes for creating complex container-bound Google Suite applications. For example, using Google Sheets:

  • CachedTable improves the read and write speeds of accessing range data by at least 80-90%.
  • TriggerManager greatly simplifies the management of simple OnEdit triggers so that you just need to provide the cell range and callback function to call whenever that range is edited.
  • Stopwatch simplifies the measurement of computation time.

Version History

v.0.0.1 Initial alpha release. Provides 3 classes, CachedTable, TriggerManager and Stopwatch.

v0.0.2 Updated Cached Table JSDoc, renamed methods to allow Gaslight as an overall library identifier. Enhanced CachedTable to return its Object reference for some methods so that chaining can take place.

Tech

Gaslight is written using ES5 syntax where needed, tested on the V8 Javascript runtime, and itself is currently not open sourced.
